# Execução
  - Requisitos:
    - GCC previamente instalado e configurado.
  - Para compilar, digite:
  
        cd ./missionarios/
        gcc queue.c main.c -o missionario
  - Para executar, basta chamar pelo arquivo no Terminal:
  
        ./missionario
    ou, pelo Prompt de Comando:

        .\missionario.exe
